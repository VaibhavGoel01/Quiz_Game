# Quiz_Game

A web-based quiz game platform with multiple quiz categories to test and enhance your knowledge.

## 📋 Description

Quiz_Game is an interactive web application that provides different quiz games to help users learn and test their knowledge in various subjects. The platform features a clean, user-friendly interface with responsive design.

## ✨ Features

- **Multiple Quiz Games**: Access different quiz categories
- **Interactive UI**: Hover effects and smooth transitions
- **Responsive Design**: Works on different screen sizes
- **Visual Feedback**: Color-coded responses (correct answers in green, incorrect in red)
- **Score Tracking**: Track your performance throughout the quiz

## 🛠️ Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling and animations
- **JavaScript** - Interactive functionality

## 🎮 Game Categories

- **Let's Prepare OS** - Test your Operating Systems knowledge

## 📝 Usage

- Click on any game card to start the quiz
- Select your answer from the available options
- Get immediate feedback (green for correct, red for incorrect)
- Complete all questions and review your score

## 🎨 UI Design

- Clean, minimalist interface
- 2-column responsive grid layout
- Card-based game selection
- Smooth hover animations
- Color-coded answer feedback system
